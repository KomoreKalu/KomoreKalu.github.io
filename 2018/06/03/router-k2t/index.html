某讯K2t刷入breed

2018-06-03

路由器固件版本 22.23.1.108

路由器管理界面登录密码设置为admin

PC连接路由器，有线或无线均可

1.开启Telnet

运行RoutAckK2TV1B1

按图示填写K2t管理界面登录地址

按图示填写K2t管理界面登录地址

点击唤醒设备

成功唤醒程序会收到设备识别码 343F6ECC3936CB73350B27A405FA6C4C

按需点击 『打开Telnet/设置永久标志』

2.开启ssh

使用PuTTY按图示登录路由器

在PuTTY中依次执行命令

echo -e 'admin\nadmin' | passwd root

/etc/init.d/dropbear start

3.上传并刷入过渡固件

运行WinSCP

按图示登录路由器

&更新

返回上一级再进入tmp文件夹

将过渡固件复制过去

上传完成后在PuTTY中执行如下命令刷入固件

mtd -r write /tmp/openwrt-k2t-initramfs-factory-uboot-unlock.bin firmware

注意：刷入固件过程中若断电会导致路由器变砖

4.开启过渡固件的ssh

路由器重启后重新连接，过渡固件的无线信号名称为NATCAP_XX，密码：88888888

浏览器访问192.168.15.1并登录 密码为： admin

依次点击 系统 管理权

并将如下三项打勾

再点击 保存&应用

5.上传并刷入breed

运行WinSCP登录路由器

将breed文件复制到tmp文件夹

运行PuTTY以ssh连接路由器

依次输入

用户名 root

密码 admin

（ps：输入密码时PuTTY中并不会显示密码）

再按回车登录

执行命令

mtd write /tmp/breed-qca9563-phicomm-k2t.bin u-boot

等待breed刷入即可
