笔记本加装固态硬盘

2018-08-23

迫于笔记本MibookPro 自带固态容量有限

遂加装一块固态硬盘 Intel Pro2500 360G

拆卸

拆后盖前笔记本*断电并关机，若空气干燥则需防范静电***

先卸下后盖上的螺丝(包括脚垫中藏的一个总共九个)

使用撬片从转轴左端开始拆

以逆时针的顺序拔开卡扣

然后拆开出风口的卡扣

将后盖平稳地向转轴方向推动

取下后盖

安装

卸下笔记本硬盘位自带的固定螺丝

将固态硬盘金手指端向下倾斜20~30° 插入接口中

按下硬盘尾端并上好螺丝固定

检验

在安回后盖前先开机检验系统能否正常读取硬盘

以及用CrystalDiskInfo或其它工具查看硬盘的通电次数
