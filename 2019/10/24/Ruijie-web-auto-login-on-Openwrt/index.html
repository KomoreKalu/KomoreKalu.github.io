在OpenWrt上自动化锐捷web认证

2019-10-24

设备信息

一只刷了Openwrt的软路由，配置如下

调试脚本

首先，执行

wget https://raw.githubusercontent.com/LGiki/RuijiePortalLoginScript/master/ruijie_template.sh

下载脚本文件

在未认证状态打开web认证界面，网站url中index.jsp?后的全部字符为之后需要的queryString参数，保存备用

执行

vi ruijie_template.sh

将脚本的base bin#!/bin/bash修改为#!/bin/sh

修改service和queryString

我校校园网仅有连通接入，故service部分留空

保存退出后执行

chmod +x ruijie_template.sh赋予脚本运行权限

opkg install curl安装curl

./ruijie_template.sh username password进行认证

（图中我将脚本文件名改为了autologin.sh）

返回信息中包含"result":"success"即是认证成功

Crontab自动执行

依次执行
echo '30 6 * * * /root/autologin.sh username password'>>/etc/crontabs/root

service cron restart
即可在每天6:30自动执行认证脚本
