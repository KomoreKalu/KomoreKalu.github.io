在Debian Bullseye上使用Fcitx5-rime

2020-12-12

长话短说

安装 sudo apt install fcitx5 fcitx5-rime fcitx5-material-color

配置 fcitx5-config-qt

添加中州韻

按需修改附加组件 > 经典用户界面

启用单行模式

vi ~/.config/fcitx5/conf/rime.conf

修改或添加

PreeditInApplication=True

在gnome-tweaks > 开机启动程序中添加Fcitx5
